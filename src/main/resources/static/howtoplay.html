Impostor Game / Játék

Játékmenet / How to Play

Magyar:

  • Mindenki kap egy szót, kivéve az Imposztor. A szóról valamilyen jellemzőt kell mondani, de úgy, hogy ne legyen egyértelmű az imposztor számára.
  • Körbe kell menni, a kört tetszőlegesen kell megkezdeni és folytatni az irányba. Mindenki mondjon egy szót. Ha gyanítod, hogy ki az Imposztor, kérvényezz szavazást.
  • Ha a többség kérvényezte, elindul a szavazás, amit a szöveg is jelez. Szavazatot kell leadni valakire. Ha valaki többségi szavazatot kap, kiesik!
  • Nyerni úgy lehet, ha a szavazás képtelen játékfeltételig jut el az Imposztor, vagy kiejtik az Imposztort.
  • Az idő lejártával folyamatos kényszerített szavazások indulnak.

English:

  • Everyone receives a word, except the Impostor. You need to say a characteristic about the word without making it obvious for the Impostor.
  • Take turns in any order. Each player says a word. If you suspect someone is the Impostor, request a vote.
  • When a majority requests a vote, it starts (indicated by text). Cast your vote for a player. If a player receives the majority, they are eliminated!
  • Win by either reaching the game’s conditions without the Impostor being discovered, or by voting the Impostor out.
  • When time runs out, forced votes occur continuously.

Technikai részletek / Technical Details

Magyar:

  • Szükséges Java verzió: Java SDK 23.0.1 vagy nagyobb.
  • 3-8 főig.
  • A résztvevőknek egy hálózaton kell lenniük, vagy nyitott port.
  • A port módosítható: jar/BOOT-INF/classes/application.properties fájlban: server.port=? (Alapértelmezett: 80)
  • Indítás: dupla kattintás vagy terminál: java -jar Impostorgame.x.x.jar (.bat)
  • Leállítás: a kezelőfelületen jobb felső sarokban a 'Leállítás' gomb.
  • Saját szavak használata: jar/BOOT-INF/classes/static/words/ mappában található .txt fájlok. Neved hozzáadása: "hu_típus.txt" formátumban.
  • A fájl szerkesztéséhez használj olyan programot, ami képes .jar fájlokat megnyitni (pl. WinRAR).

English:

  • Required Java version: Java SDK 23.0.1 or higher.
  • Supports 3-8 players.
  • Players must be on the same network or open port.
  • Port can be modified in jar/BOOT-INF/classes/application.properties: server.port=? (Default: 80)
  • Start: double click or terminal: java -jar Impostorgame.x.x.jar (.bat)
  • Stop: Use the 'Shutdown' button at the top-right of the control panel.
  • Custom words: Add .txt files to jar/BOOT-INF/classes/static/words/. File name format: "en_type.txt" or "hu_type.txt".
  • To edit the .jar files, use a program that can open them, e.g., WinRAR.
